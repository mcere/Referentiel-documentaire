Elle définit :

- les procédures de validation ;
- les mécanismes d'attribution ;
- les formats documentaires ;
- les règles de présentation ;
- les mécanismes de conformité ;
- les règles d'exécution pour IA.

# Principes directeurs

L'application du référentiel repose sur les principes suivants :

## Vérifiabilité

Une information importante doit pouvoir être vérifiée.

---

## Exactitude

Les conclusions doivent demeurer compatibles avec la documentation disponible.

---

## Traçabilité

L'origine documentaire d'une information doit pouvoir être identifiée.

---

## Transparence

Les limitations doivent être explicites.

---

## Complétude documentaire

Le niveau de documentation doit être indiqué.

---

## Uniformité

Les unités d'un même corpus doivent respecter les mêmes exigences.

---

## Réutilisabilité

Les résultats doivent pouvoir être convertis, cités et réemployés.

---
