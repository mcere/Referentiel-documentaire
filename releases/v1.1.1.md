### Changement technique majeur

Le script de compilation ne dépend plus d'une liste codée en dur des documents normatifs.

La compilation repose maintenant sur les références présentes dans :

`MANIFEST.md`

Ce changement permet :

- l'ajout futur de nouveaux documents sans modification du script ;
- une meilleure cohérence documentaire ;
- une gouvernance centrée sur le manifeste.
