# Index du Référentiel documentaire

## Avertissement

INDEX.md est un document de navigation.

Le contenu normatif officiel du Référentiel documentaire est défini par :

- MANIFEST.md

Le manifeste prévaut toujours en cas de divergence.

Aucun contenu de INDEX.md ne doit être interprété comme modifiant les documents normatifs du référentiel.

# Règle de priorité documentaire

En cas de conflit :

1. Le contenu de MANIFEST.md prévaut.
2. Les documents normatifs contenus dans `docs/` prévalent.
3. Les annexes complètent les documents normatifs.
4. README.md, INDEX.md, VERSION.md et CHANGELOG.md n'ont pas de valeur normative.

---

# Référentiel canonique

Le Référentiel documentaire canonique correspond à l'ensemble des documents identifiés dans :

`MANIFEST.md`

Aucun document non mentionné dans le manifeste ne fait partie du référentiel canonique.
